## 1. 프로젝트 개요

### 1.1 무엇이 바뀌는가

Contract Sync v1(계약서 중심 도구)을 **고객·업무 중심 프로젝트 관리 플랫폼**으로 전면 재설계한다.

```
[v1] 계약서 업로드 → AI 분석 → 업무 자동 생성

[v2] 발주처 등록 → 프로젝트 → 업무 수행 → 완료 보고 → 피드백 → 수금
                      ↑ 문서(계약서/견적서)는 "첨부" 역할
```

### 1.2 핵심 변경 요약

| 항목 | v1 | v2 |
|------|----|----|
| 최상위 엔티티 | 계약서 (Contract) | 발주처 → 프로젝트 → 업무 |
| 업무 저장 방식 | contracts.tasks JSON | tasks 독립 테이블 |
| 프로젝트 유형 | 외주만 | 외주 / 내부 / 유지보수 |
| 문서 역할 | 핵심 (필수) | 첨부 (선택) |
| 발주처 관리 | 문자열 필드 | clients 독립 테이블 |
| 업무 완료 후 | 상태만 변경 | 완료 보고 → 피드백 → 확인 |
| 보고서 | 없음 | AI 자동 생성 (일간/주간/월간) |
| 수금 관리 | 없음 | 결제 일정 + 알림 |

### 1.3 기술 스택

| 구분 | 기술 |
|------|------|
| 백엔드 | FastAPI (Python 3.12), SQLite (aiosqlite), SQLAlchemy 비동기 |
| 프론트엔드 | HTML5 + Tailwind CSS + Alpine.js 3.x (SPA) |
| AI 엔진 | Google Gemini 2.0 Flash |
| 이메일 | aiosmtplib + Jinja2 템플릿 |
| 외부 연동 | Google Sheets API v4, Google Calendar API |

# app/api/router.py — v2 최종
from app.api.endpoints import (

    completion_reports, feedbacks,       # Phase 2

api_router = APIRouter()
